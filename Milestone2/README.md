# Milestone 2 – PCB Defect Detection

## Objective
Classify PCB defects using a CNN model.

## Defect Classes
- missing_hole
- mouse_bite
- open_circuit
- short
- spur
- spurious_copper

## Model
- EfficientNetB0
- Image size: 128x128

## Outputs
- Trained model
- Confusion matrix
- Accuracy/Loss plots
- Annotated inference images
